实验器材:
	潘多拉STM32L475开发板
	
实验目的:
	学习STM32待机唤醒功能
	
硬件资源:
	1,RGB灯
	2,WK_UP按键
	3,TFTLCD
	
	
实验现象:
	本实验通过长按（3秒以上）WK_UP按键开机，并且通过RGB的闪烁指示程序已经开始运行，再次长
	按该键，则进入待机模式，RGB关闭，程序停止运行。类似于手机的开关机。
	
注意事项:
	如果MCU处于待机模式下，ST LINK是识别不到芯片的，也就是调试和下载都使用不了，必须
	进入正常运行模式才可以下载程序。
